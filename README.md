# porwigle-cms

It's simplistic cms inspired by frogcms. Work really much in progress.

## Usage

Kick repl in and

```
user=> (load-test-site)
user=> (go)
```

And you should see something at http://localhost:8081/
Admin ui is at http://localhost:8081/admin/index.html
